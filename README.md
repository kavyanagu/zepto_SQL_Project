**Zepto Inventory Analysis using SQL**

This project focuses on analyzing Zepto’s inventory dataset using SQL to derive meaningful business insights related to product availability, pricing, and revenue.

**Project Overview**

The SQL script creates and manages an inventory database containing product details such as category, price, discount, stock availability, and quantity. It performs data exploration, cleaning, and analysis to support data-driven decision-making.

**Key Features**

Database Setup: Created a structured table with relevant product attributes.
Data Exploration: Queried total records, sample data, null values, and unique product categories.
Data Cleaning: Removed invalid records (e.g., zero pricing) and standardized price units.
Stock Analysis: Compared in-stock vs out-of-stock products and identified duplicate product entries.
Business Insights:
  Top discounted (best-value) products
  High-value products that are out of stock
  Category-wise estimated revenue
  Products with high MRP and low discounts

**SQL Techniques Used**

Aggregations (SUM, COUNT)
Filtering (WHERE, HAVING)
Sorting (ORDER BY)
Grouping (GROUP BY)
Data manipulation (UPDATE, DELETE)
Distinct queries for deduplication

**Outcome**

The project helps understand inventory performance, pricing strategies, and stock management, enabling better restocking and revenue optimization decisions.
